# My-chef

## 개발 배경

코로나 시대!
접촉 피하기 위한 '언택트 사회' 등장  
외식이 급감하고 배달 음식 소비가 증가  
건강한 식생활과 코로나 블루 극복을 위해 집 있는 재료로 요리해서 나를  
위한 한끼를 만들어보자!!!!!  
우리 집 냉장고 속 재료를 분석해서 만들 수 있는 음식을 추천해 드립니다.

## contents

1. 음식 재료 인식 모델
2. 재료를 활용한 레시피 추천
3. SNS 기능
